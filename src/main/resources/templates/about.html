About Our API

A RESTful API for task management

Overview

This API provides endpoints for managing tasks in a simple and efficient way. Built with Spring Boot, it offers full CRUD operations with data persistence using MySQL.

Available Endpoints

Method Endpoint Description
GET /api/tasks List all tasks
POST /api/tasks Create a new task
PUT /api/tasks/{id} Update an existing task
DELETE /api/tasks/{id} Delete a task
