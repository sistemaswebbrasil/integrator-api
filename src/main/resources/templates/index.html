Integrator API

Sistema de gestão e integração de serviços

Sobre o Projeto

Esta é uma API de demonstração desenvolvida para a plataforma Integrator Host do Brasil, focada na implementação de operações CRUD (Create, Read, Update, Delete) utilizando as melhores práticas de desenvolvimento.

Visite Integrator Host Documentação API

Stack Backend

  • Spring Boot 3.2.x
  • Java 17
  • PostgreSQL
  • JPA/Hibernate

Recursos da API

  • Arquitetura REST
  • Documentação Swagger/OpenAPI
  • Deploy automatizado
  • Testes unitários
  • Validação de dados

Aplicação Frontend

Esta API é consumida por uma aplicação frontend desenvolvida em Nuxt.js e hospedada na Vercel. A aplicação fornece uma interface amigável para interagir com os endpoints desta API.
